# Featured Projects

## LLaMA Alignment Pipeline (SFT + DPO) on Apple Silicon

**Technologies:** Python, MLX, MLX-LM, LoRA, DPO, Hugging Face

Developed an end-to-end alignment pipeline for fine-tuning LLaMA models using Supervised Fine-Tuning (SFT) and Direct Preference Optimization (DPO), optimized for Apple Silicon hardware. The project includes LoRA adapter training, model merging, interactive chat interfaces, and deployment-ready export formats.

**Highlights**

* Native Apple Silicon training using MLX
* LoRA-based parameter-efficient fine-tuning
* DPO alignment workflow
* Export support for Hugging Face, OpenAI-compatible APIs, and GGUF

**Repository:** [github.com/AdityaInamdar334/llama-mlx-alignment](https://github.com/AdityaInamdar334/llama-mlx-alignment)

## RSSM World Model (Dreamer-Style Reinforcement Learning)

**Technologies:** PyTorch, NumPy, MPS, Matplotlib, HTML/CSS/JavaScript

Implemented a Dreamer-inspired Recurrent State Space Model (RSSM) trained on pixel observations from the DeepMind Control Suite. The project includes world model training, open-loop imagination rollouts, diagnostics, and an interactive visualization dashboard.

**Highlights**

* 5M-parameter RSSM implementation
* 27.4× training acceleration using Apple Silicon MPS
* Stable long-horizon latent rollouts
* Interactive web-based evaluation dashboard

**Live Demo:** [adityainamdar334.github.io/cheetah-rssm-world-model](https://adityainamdar334.github.io/cheetah-rssm-world-model/)  
**Repository:** [github.com/AdityaInamdar334/cheetah-rssm-world-model](https://github.com/AdityaInamdar334/cheetah-rssm-world-model)
